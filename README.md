# Мобильное приложение "Турагентство"

![App Icon](/images/5AHQ7n1TYVE.jpg)

### Краткое описание:
Мобильное приложение для туристического агентства, которое позволяет пользователям просматривать туры, оформлять заказы и просматривать статистику. Приложение предназначено для двух типов пользователей: туроператоров и клиентов. Туроператоры могут добавлять и редактировать туры, а также просматривать статистику. Клиенты могут просматривать доступные туры и оформлять заказы.
